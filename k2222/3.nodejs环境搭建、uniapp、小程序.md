## 二.vue的脚手架安装
```javascript
npm install -g @vue/cli
```

```javascript
// 查看是否安装成功
vue --version
```

## 三.创建vue项目
```JavaScript
# npm 6.x
npm init vite@latest <project-name> --template vue

# npm 7+，需要加上额外的双短横线
npm init vite@latest <project-name> -- --template vue

cd <project-name>

npm install

npm run dev
```

> 微信小程序项目直接使用HBulider X新建一个uni-app项目，再运行至微信开发者工具直接就是微信小程序项目了
